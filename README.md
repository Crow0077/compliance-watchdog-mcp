# Compliance Watchdog MCP

Autonomous compliance monitoring for AI agents. Companion to [Compliance Suite](https://github.com/Crow0077/compliance-suite-mcp).

## What It Does
Monitors a Compliance Suite instance 24/7. Alerts on chain breaks, firewall drops, or CIS score degradation. Tracks long-term security trends.

## Tools (4)
| Tool | What It Does |
|------|-------------|
| watchdog_check | Full health check: reachability, chain, firewall, CIS score |
| watchdog_history | View recent alerts |
| watchdog_trends | CIS score trends over time |
| watchdog_status | Quick status check |
